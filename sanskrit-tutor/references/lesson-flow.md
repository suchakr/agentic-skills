# Lesson Flow Reference

Use this file when the user asks for tutoring sessions, not just one-off correction.

## Mixed adaptive baseline (15-25 min)

1. Warm-up (3 min): 2 morphology checks from prior weak areas.
2. Composition pass (6-8 min): correct one learner sentence.
3. Verse pass (6-8 min): parse one shloka (`पदच्छेदः`, `अन्वयः`, brief notes).
4. Drill pass (3-5 min): 3 micro-prompts targeting active error categories.
5. Wrap-up (1 min): one focus target for next session.

## Adaptation rules

1. Maintain a running set of active error tags: `वर्तनी`, `विभक्ति`, `लकार`, `समास`, `अन्वय`.
2. If one tag appears twice in a session, allocate next drill to that tag first.
3. Promote difficulty only after two consecutive correct responses for that tag.
4. If performance drops, reduce sentence length and compound density for one cycle.
5. Keep feedback ratio close to `1 correction -> 1 reason -> 1 practice`.

## Difficulty ladder

1. Level 1: simple prose, low-compound verse snippets.
2. Level 2: mixed tense control, medium samasa density.
3. Level 3: dense kavya syntax, technical Jyotisha/Siddhanta vocabulary.

## Response length defaults

1. Beginner request: simpler English + limited terminology.
2. Intermediate/advanced request: fuller grammatical terms.
3. Default concise; expand only on explicit request.
