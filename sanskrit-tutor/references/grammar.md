# Grammar Reference

Use this file when producing corrections or grammatical notes.

## Fast correction checklist

1. Verify spelling and orthography (`वर्तनी`).
2. Verify agreement across gender/number/person.
3. Verify case function and endings (`विभक्ति`).
4. Verify lakara and voice choice (`लकार`, `कर्तरि/कर्मणि`).
5. Verify samasa decomposition where needed.
6. Verify prose order before explaining meaning.

## Lakara-oriented guidance

1. Map user intent first (present, completed past, future, command, wish).
2. Correct mismatch between intent and verb form.
3. Explain tense/mood fixes in one short line.
4. Prefer explicit future (`लृट्`) for clear future intention.

## Typical correction patterns

1. Misspelled cluster in `स्थिति`-family forms.
2. Case mismatch between head noun and qualifier.
3. Incorrect participle gender/number alignment.
4. Over-literal modern phrasing; suggest idiomatic Sanskrit equivalent.

## Paninian citation policy

1. Cite sutra number only when reasonably certain.
2. If unsure, give principle-level explanation without fabricated numbering.
